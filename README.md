# 🩺 Diabetes Prediction Using Logistic Regression

## 📌 Project Overview
This project predicts whether a person is diabetic using the Logistic Regression algorithm. The model is trained on the Kaggle Diabetes Dataset after preprocessing, feature scaling, and exploratory data analysis.

## 🎯 Objectives
- Predict diabetes using Machine Learning.
- Perform data preprocessing and feature scaling.
- Visualize the dataset using graphs.
- Train and evaluate a Logistic Regression model.

## 🛠️ Technologies Used

- Scikit-learn
