# Diabetes Prediction Using Logistic Regression

## Project Overview
This project predicts whether a person is likely to have diabetes using the Logistic Regression machine learning algorithm. 
The model is trained on a diabetes dataset and predicts the outcome based on medical parameters such as glucose level, BMI, age, insulin, blood pressure, and other health-related features.

## Objectives
- Analyze the diabetes dataset.
- Perform data preprocessing and cleaning.
- Train a Logistic Regression model.
- Evaluate the model using accuracy and confusion matrix.
- Predict diabetes for new patient data.
